## 🏘️ 프로젝트 개요

소소(SOSO)는 **지역 참여 커뮤니티**를 통해 주민들의 목소리를 듣고, **창업 아이템 제안 및 리포트**를 제공하며, **포인트 교환 상점**을 운영하는 통합 플랫폼입니다.
