  </Card>
</CardGrid>

### 핵심 기능

  <Card title="🐻 기본 스토어">
    **간단한 상태 생성**
    ```tsx
    const useStore = create((set) => ({
      count: 0,
      increment: () => set((state) => ({ 
        count: state.count + 1 
      }))
    }));
    ```
  </Card>
  
  <Card title="💾 영구 저장" >
    **persist 미들웨어**
    ```tsx
    const useAuthStore = create(
      persist(
        (set) => ({ user: null }),
        { name: 'auth-storage' }
      )
    );
    ```
  </Card>
  
  <Card title="🎯 선택적 구독" >
    **성능 최적화**
    ```tsx
    // 특정 값만 구독
    const userName = useStore(state => state.user?.name);
    
    // 여러 값 구독 (shallow 비교)
    const { user, isLoading } = useStore(
      state => ({ user: state.user, isLoading: state.isLoading }),
      shallow
    );
    ```
  </Card>

  <Card title="🔧 미들웨어">
    **확장 기능**
    ```tsx
    const useStore = create(
      devtools(
        persist(
          immer((set) => ({ /* state */ })),
          { name: 'store' }
        )
      )
    );
    ```
  </Card>

## 🎭 유틸리티 라이브러리들

### clsx + tailwind-merge

<CardGrid>
  <Card title="🔀 clsx" >
    **조건부 클래스 결합**
    ```tsx
    import { clsx } from 'clsx';
    
    clsx(
      'base-class',
      condition && 'conditional-class',
      { 'object-class': isActive }
    );
    ```
    
    **장점:** 직관적인 조건부 로직  
    **단점:** Tailwind 중복 처리 안됨
  </Card>

  <Card title="🔄 tailwind-merge" >
    **Tailwind 클래스 병합**
    ```tsx
    import { twMerge } from 'tailwind-merge';
    
    // 중복 클래스 자동 해결
    twMerge('px-2 py-1 px-4') // 결과: 'py-1 px-4'
    
    // 조합 사용
    const cn = (...inputs) => twMerge(clsx(inputs));
    ```
    
    **장점:** Tailwind 충돌 해결  
    **단점:** 번들 크기 약간 증가
  </Card>
</CardGrid>
